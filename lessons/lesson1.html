Print Statements and Basic Data Types in C

 Introduction

Welcome to the first lesson on C. On this page, you will learn about the syntax of C, simple data types, how to compile C code, how to print in the terminal, and how to define a variable. This lesson will be split up into subsections so feel free to skip around.

First lets start of with something relatively simple and that is printing a statement in C. You can follow along if you want by making a new C file and typing in the C code we are about to write. The first thing we need to do is write a main function which is what your program will run first. The function will look like...

    #include <stdio.h>
    int main() {
    printf("%s","Hello World\n");
    return 0;
 }

Because this is a very simple program, we will only use the main but note that you can create other functions that can be called in main. For more information about functions, visit our functions lesson. Also don't worry about what the include statement or what the %s is doing, just know that we need it in order to use the print function and to print a string in C. Also to note that the "\n" indicates that we want to print a new line after "Hello World". We can then run it by opening our terminal and cd all the way to our program. Once we are in the folder of our program we can type this into our terminal...(note you must have gcc installed for this to work) 

gcc -g -std=c99 -Wall -pedantic -o insertAnyName yourNameOfYourFile.c

What the line above is doing is it is calling the gcc compiler and passes the -g flag for "debugging" which you can learn more on our debugging lesson.

Then we pass in the -std=c99 flag to tell gcc that we want to compile a c99 program. We then call -Wall and -pedantic to tell gcc that we want to look out for all errors in our code and all warnings in our code as well. Finally we call the -o flag and pass in the parameters insertAnyName and yourNameOfYourFile.c which converts your c code with the flags/preferences you passed in, and turn it into an object file called insertAnyName in which you can finally run as the user.

We finally run the program by typing in this in our terminal

./insertAnyName

After doing this you should get the output of "Hello World" on the terminal. Congrats you made your first program! Below are more examples of how printf can be used.

Print Statements using an integer and a string

Above we printed a string of "Hello World". If you wanted to print an integer and a string together, we can edit the above code to say...

    #include <stdio.h>
    int main() {
    printf("My favorite number is %d\n", 94);
    return 0;
 }

Again to note a %s is equal to a string format and %d is equal to a decimal/integer formatter which is how C knows which type we want to print. We will talk more about why data types like strings and integers are so important to memorize later on but for right now just know that this is how you print in C. %s was not needed for the example above as we did not use a variable to concatenate a string but for a more complex program you will see why %s is needed.

 Simple Data Types

Characters

A character in C is a single byte like a number, symbol or a letter. For more information about bytes visit the byte lesson for more information.

 Strings

If you did the following exercise above you might have recognized why we did "" on the printf statement. The reason that is needed is because the "" means that the letters inside we are dealing with are a String datatype. A String datatype is an array of characters that ends with a null terminator of '\0'. This terminator is important as it tells the program the string has ended and it does not need to look further for more data. This fine detail will be important for lessons later on for other functions like strlen (string length) and scanf.

 Integers

An integer just like in math is a whole number that can be positive, negative, or a zero. As long as it does not have a decimal, the number is stated to be a integer datatype. Example include 100, 234, and -38462. Warning: integers can be strings if you allow it to be, be careful on where you wrap your quotations. If they are wrapped in quotations, they will follow the rules of strings and you will not be able to do math with them.

 Doubles

A double is a number that contains a decimal point. They can contain up to 15 digits in total and you can do math with this type. If you want to print a double, you can do that by using the %f format specifier when you are using printf

 Variables

Like all programing languages, a variable is a storage location that is paired with a name which stores data that can be known or unknown to the programmer. Think of it as storage container that can update as the program runs and hits certain cases like an if statement. If you want to create a variable named myNumber that stores a integer, you can do that by doing...

    #include <stdio.h>
    int main() {
    int myNumber = 2;
    return 0;
 }

You can then easily print that number with a new line character by doing...

    #include <stdio.h>
    int main() {
    int myNumber = 2;
    printf("My favorite number is %d\n", myNumber);
    return 0;
 }

The "\n" represents a new line character and it is important to include it especially if you are printing multiple print statements as you always want to space them out. If you want to create a variable that stores a string and a double that gets printed, you can do that by doing...

    #include <stdio.h>
    int main() {
    double myDecimalNumber = 2.0;
    char str[] = "inches"
    printf("I grew %f, %s\n", myDecimalNumber, str);
    return 0;
 }

For assigning a string variable in C it is important to allocate the amount of space the string will take up. In the above example, we can see that we did not do this as it was not needed because we were directly assigning it to to the variable called string right away. But what happens if we wanted to make an empty string variable that we will not use until later on. This example I described would look like...

    #include <stdio.h>
    int main() {
    // We put 25 inside the brackets so we want to allocate space for at least a 25 string word. This number can be any integer
    so long as it at least fits the string length you want to store always try and aim for the smallest number that
    will fit the the string you want to save
    char str[25];
    // We do random stuff here but after we do this random stuff...
    // How do we assign a empty variable called str to a string here?
    return 0;
 }

If we wanted to assign the variable str after it was assigned as an empty value or "NULL" (which means empty/zero value) we can use the function strcpy (string copy) to overwrite the empty value into a value we want. The above example will now look like... 

    #include <stdio.h>
    #include <string.h>
    int main() {
    char str[25];
    strcpy(str,"Apple");
    printf("%s\n",str);
    return 0;
 }
