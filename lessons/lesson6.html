The C Standard Library

 Introduction

In this section you will learn about the C library and the functions it has to offer. We will only go over the main ones almost every programmer uses. For more information about C library's we will not be covering in this lesson check out the official C library website for more information.

 Uses

The C Standard library is a group of functions already pre-made by the developers of the languages of C that you may freely use. Because of this, it is on us as developers to use these as they are stress tested, reliable, and (almost mostly) bug free. Below is a chart of the library's you can use and implement for your programs. These of course are not all the library's C has and the chart below is the future library's we will use. To call a library place <libraryNameHere> on top of the file you are programming in. These library's do not transfer to other files in your project so always make sure to import them on top of the files you are working on.

 The library's we will be using for future lessons

Library Description
<string.h> Use this library when you want to manipulate a string (array of characters)
<ctype.h> Use this library when you want to test and map certain characters
<math.h> Use this library when you want to use built in formulas for math
<stdio.h> Use this library when you want to manipulate, scan, or edit a file using system arguments
<stdlib.h> Use this library when you want to dynamic memory methods for your program

Examples of some <string.h> functions

Function Description
strcpy copies one string from source and overwrites the value of the intended destination
strcat copies one string from source appends it to the end of the intended destination
strcmp compares the source string and the destination string and compares them alphabetically. (Good for checking if strings are equal)
memset takes a source and a size and is used to clear source of "junk" inside source.
strlen returns the count of characters in source, the function stops counting when it sees it's first '/0'

Examples of some <math.h> functions

Function Description
sqrt finds the square root of the source
sin computes the sine of the source
pow computes the pow of the source based on the second arg of what power you pass in
log10 computes the natural log of source
floor takes the source you pass in and rounds down. EX: floor(5.7) = 5

Examples of some <stdio.h> functions

Function Description
printf takes the source and prints to stdout
fclose closes the file/stream
getc gets the next character from the file you are reading
remove deletes the given file you pass into source
fopen opens the file you pass into source using the given mode you pass as into the second argument

Examples of some <stdlib.h> functions

Function Description
malloc Allocates memory and returns a pointer for later use.
free destroys the allocated memory that was made from malloc, calloc, or realloc.
calloc Allocates memory, clears junk, and returns a pointer for later use. (Slower than malloc)
realloc resizes the memory already allocated by malloc or calloc. Takes in the the memory allocated, and the size you want to resize it to
exit terminates the running program
